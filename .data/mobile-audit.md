## Instructions

**When performing mobile audit:**

1. Open DevTools and set viewport to 375px × 667px (iPhone SE)
2. Navigate to each page
3. Document issues in this format:

```markdown
- [ ] **Page:** `/jobs` **Issue:** Filter sidebar doesn't collapse **Severity:** High **Fix:** Implement mobile drawer component
```

4. After fixing, check the box and add " - **Fixed:** [commit hash or date]"

**Severity levels:**
- **Critical:** Breaks functionality or makes page unusable
- **High:** Major UX issue but page remains usable
- **Medium:** Minor annoyance, fixable
- **Low:** Polish or nice-to-have
